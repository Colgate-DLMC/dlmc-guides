# Adding Captions in Premiere

Quick reference for adding closed captions to a Premiere Pro project.

## Step 1: Import your transcript

Open the **Text** panel (Window > Text) and click **Transcript**.

## Step 2: Sync captions to timeline

Once transcribed, click **Create Captions** to generate a caption track.

!!! tip
    If auto-sync drifts, right-click the caption track and select
    **Re-sync Captions**.

!!! warning
    Exported captions default to .srt — confirm the LMS/platform
    you're uploading to actually supports that format.
